* Advocate profile & biography
* Practice areas
* Case expertise
* Contact information

* Responsive UI design

The previous version has been modernized using **Vite** for faster builds and better development experience.

##  Installation & Setup

Clone the repository:

```bash
git clone https://github.com/ShakilUrRehman21/lawWebsite.git
```

Navigate to project:

```bash
cd sarfaraz-law
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
